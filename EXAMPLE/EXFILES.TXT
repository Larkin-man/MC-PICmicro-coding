---------------------
Example Files
---------------------
The source code for examples used in Microchip Technology Development
Tools user's guides is generally provided in the document.  However,
some example code, due to size or the possibility of frequent change,
is not printed. Therefore, this code is provided as part of the 
MPLAB IDE installation.

In addition, other undocumented example files may be included in the
installation.

These example files are discussed here.
---------------------
Примеры файлов
---------------------
Исходный код для примеров, используемых в руководствах пользователя Microchip Technology Development Tools, обычно предоставляется в документе. Однако некоторые примеры кода из-за размера или возможности частого изменения не печатаются. Поэтому этот код предоставляется как часть установки MPLAB IDE.

Кроме того, в установку могут быть включены другие недокументированные файлы примеров.

Эти примеры файлов обсуждаются здесь.
